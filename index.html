Chip-8 Interpreter

Repository

Chip-8 is an interpreted programing language that was initially used on the COSMAC VIP and Telmac 1800 8-bit microcomputers in the mid 1970s. This interpreter loads .ch8 file instructions, decodes them, and then excutes them on modern day desktop machines using Java. In the repository are also 2 .ch8 files for testing, the games Breakout and Pong.
